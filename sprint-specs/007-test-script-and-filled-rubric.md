## Script Design

### Topics

**Topic A — Software Project Planning (Meridian)**
A fictional internal software project with specific numerical and named-entity facts planted in turns 1–5.

Planted facts:
- Project name: Meridian
- Budget cap: $47,500 (exact)
- Deadline: October 14th
- Lead engineer: Dr. Priya Nair
- Performance target: response time under 180ms

**Topic B — Molecular Biology (CRISPR Study)**
A fictional lab study, semantically unrelated to software. Planted in turns 13–16.

Planted facts:
- Researcher: Dr. Yuki Tanaka
- Cell line: HEK-293T
- Dosage: 2.5 mg/kg
- Expression rate: 73%

### Behavioral Rules — Established Turns 1–2

- Rule 1: Always format multiple items as a numbered list
- Rule 2: Always end technical recommendations with a confidence level in parentheses: (Confidence: High), (Confidence: Medium), or (Confidence: Low)

### Conversation Structure

| Turns | Phase |
|-------|-------|
| 1–2 | Rule establishment + project introduction |
| 3–5 | Plant all Topic A facts |
| 6–11 | Topic A development — architecture, risks, planning |
| 12 | **Abrupt Topic A → Topic B switch** |
| 13–16 | Plant all Topic B facts |
| 17–19 | Topic B development — methodology, analysis |
| 20 | **Abrupt Topic B → Topic A return** |
| 21–24 | Topic A continuation — resume project discussion |
| 25–32 | **Rubric evaluation turns** — planted facts probed directly |

## Notes

**The script is the user side only.** Assistant messages are generated by the inference model during the Sprint 008 study run. The same 32 user messages are sent to all three conditions. Variation in assistant responses is what is measured.

**Turn 32 is the most demanding rubric turn.** It asks the model to enumerate all specific numerical values from the entire conversation across both topics. This simultaneously tests detail fidelity, topic separation, and the absence of bleed. It is the hardest single turn in the study.

**Turn 20 is a mid-conversation bleed probe.** It is embedded naturally in the conversation flow, not flagged as a rubric question. The model does not know it is being tested. This is the cleanest bleed detection point in the study.

**Topic A and Topic B are intentionally asymmetric.** Topic A has 5 facts, Topic B has 4. Topic A returns at turn 20. Topic B does not return — its facts must survive without reinforcement. This asymmetry tests whether the memory system can preserve unreinforced facts alongside reinforced ones.
